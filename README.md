# 🗂️ نظام إدارة المهام (Task Management System)

مشروع ويب بسيط لإدارة المهام اليومية تم تطويره باستخدام **Laravel**، يتيح للمستخدم إضافة المهام، تعديلها، حذفها، وعرضها، بالإضافة إلى تغيير حالة المهمة (مكتملة / غير مكتملة).

---

## 🚀 المميزات

* إضافة مهمة جديدة
* تعديل مهمة موجودة
* حذف مهمة
* عرض جميع المهام
* تغيير حالة المهمة (pending / done)
* واجهة بسيطة باستخدام HTML و CSS

---

## 🛠️ التقنيات المستخدمة

* Laravel 12
* PHP ≥ 8.2
* MySQL
* Blade Template Engine
* CSS

---

## 📦 المتطلبات

قبل تشغيل المشروع تأكد من توفر:

* PHP ≥ 8.2
* Composer
* MySQL
* سيرفر محلي (XAMPP أو غيره)
* متصفح حديث (Chrome, Firefox, Edge)

---

## ⚙️ طريقة التثبيت والتشغيل

### 1. استنساخ المشروع

ثم عدّل بيانات قاعدة البيانات داخل `.env`في حال استخدمت قاعدة بيانات خاصة بك او سيرفر محلي غير الذي تم استخدامه:

```env
 DB_CONNECTION=mysql
 DB_HOST=127.0.0.1
 DB_PORT=3306
 DB_DATABASE=TASKS
 DB_USERNAME=root
 DB_PASSWORD=
```

---

### 4. إنشاء قاعدة البيانات

* افتح phpMyAdmin
* أنشئ قاعدة بيانات باسم:

```
task
```

---
### 5. استورد قاعدة البيانات 

### 6. تشغيل Migration 
```bash
php artisan migrate
```

---

### 7. تشغيل السيرفر
```bash
php artisan serve
```

---
### 8. فتح المشروع

افتح المتصفح:

```
http://127.0.0.1:8000/task
```

---

## 🧠 طريقة الاستخدام

* **إضافة مهمة:** عبر النموذج في الصفحة الرئيسية
* **عرض المهام:** تظهر في جدول
* **تعديل مهمة:** زر "تعديل"
* **حذف مهمة:** زر "حذف"
* **تغيير الحالة:** زر "تغيير الحالة"
